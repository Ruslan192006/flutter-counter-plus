# Flutter Counter Plus

Расширенная версия счетчика с дополнительными функциями и сохранением данных.

## Функциональность
- Увеличение и уменьшение счетчика
- Сброс счетчика
- Установка пользовательского значения
- Сохранение значения между запусками
- История изменений
- Темная/светлая тема

## Технологии
- Flutter
- SharedPreferences
- State Management (setState)
- Material Design

## Запуск
```bash
flutter pub get
flutter run
```
